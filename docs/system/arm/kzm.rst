Kyoto Microcomputer KZM-ARM11-01 (``kzm``)
==========================================

The ``kzm`` board emulates the Kyoto Microcomputer KZM-ARM11-01
evaluation board, which is based on an NXP i.MX32 SoC
which uses an ARM1136 CPU.

Emulated devices:

- UARTs
- LAN9118 ethernet
- AVIC
- CCM
- GPT
- EPIT timers
- I2C
- GPIO controllers
- Watchdog timer
